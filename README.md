## Installation

To install and run the Grocery Back-Office App, follow these steps:
1. Clone the repository:

```bash
git clone https://github.com/Mohammed4mach/grocery-back-office-app.git
```

2. Install dependencies

```bash
npm install
```

3. Run the project (using [GNU Make](https://www.gnu.org/software/make/#download))

```bash
make
```

or by using npm

```bash
npm run dev
```

4. For building the app, use

```bash
make build
```

or use

```bash
npm run build
```
